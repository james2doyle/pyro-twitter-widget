pyro-twitter-widget
===================

A twitter widget for PyroCMS. Supports oAuth v1.1 API.

### Supported REST API Endpoints

* statuses/mentions_timeline
* statuses/user_timeline
* statuses/home_timeline
* statuses/retweets_of_me
* favorites/list
